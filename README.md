## 💻 Setup

### GIT

1. Clone the repository

```bash
git clone https://github.com/floriansalvi/LAB_ThreeJsIntroduction.git
cd threeJSIntro
```

2. Dependencies

```bash
npm install
```

3. Start the server

```bash
npm run dev
```
